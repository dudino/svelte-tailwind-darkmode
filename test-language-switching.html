Language Switching Test

Open the browser console and visit your deployed app to test language switching:

  1. Visit: https://affinity-n6ywnrorf-rudys-projects-f917f7cb.vercel.app
  2. Open browser DevTools (F12)
  3. Try switching languages using the language selector
  4. Check that the URL, content, and localStorage update correctly

Manual Test Script

Copy and paste this into the browser console on your deployed app:


// Test language switching manually
console.log('Testing language switching...');

// Check current state
console.log('Current URL:', window.location.href);
console.log('Current localStorage language:', localStorage.getItem('preferred-language'));

// Test switching to Czech
localStorage.setItem('preferred-language', 'cs');
window.location.reload();

// After reload, test Russian
setTimeout(() => {
    localStorage.setItem('preferred-language', 'ru');
    window.location.reload();
}, 5000);

// After reload, test English  
setTimeout(() => {
    localStorage.setItem('preferred-language', 'en');
    window.location.reload();
}, 10000);

Expected Results

  • ✅ Content should change language immediately
  • ✅ Language selector should show the correct active language
  • ✅ localStorage should store the selected language
  • ✅ Page refresh should maintain the selected language
