Language Switching Test

Open browser console and go to: http://localhost:5177/

Console Commands to Test:

// Check current locale
console.log('Current locale:', window.location.href);

// Test localStorage
localStorage.setItem('preferred-language', 'cs');
location.reload();

// After reload, test Russian
localStorage.setItem('preferred-language', 'ru');
location.reload();

// Test English
localStorage.setItem('preferred-language', 'en');
location.reload();

What to Look For:

  • Check console for any i18n debug messages
  • Check if translations load properly
  • Check if locale changes are reflected in the UI
  • Check localStorage persistence
